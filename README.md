Vending Machine Kata
=========================
[Instructions](https://github.com/guyroyse/vending-machine-kata)

####To run tests in Java project:

######Go to project: 

    cd path/to/project

######Run tests: 

    ./gradlew test

####To run tests in Python project:

######Install nose:

    pip install nose

######Go to project: 

    cd path/to/project
######Run tests: 

    nosetests
